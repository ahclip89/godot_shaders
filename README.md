# Godot Shaders

I taught myself how to use Godot shaders and made a liquid maker.

### It has sliders to customize:
- Red Value 🟥
- Green Value 🟩
- Blue Value 🟦
- Alph Value 👻
- Wave Speed 💦
- Wave Amplitude 🌊

It will then feed those values to a shader that will render them to a square at the bottom.
